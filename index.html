API Name API Description Input
1. Cities and Placenames This "cities" API will return data about the nearest cities to the coordinates provided. In this app, I used it to output the name and the population of the first city found. The input is populated with default values which can be changed.
2. Wikipedia Fulltext Search The "Wikipedia" API will return search results from the wikipedia site. In this app, I used it to output the summary of the first search result. The default value is London, but it can be changed.
3. Postal Code Country Info The "Postal Code" API will return data about the postcodes of 95 countries. In this app, I used it to get the country code and country name of the first seven countries.
